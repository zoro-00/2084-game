# 2048 Game

A modern Android implementation of the classic 2048 puzzle game.

## Overview

This project is an Android application that brings the addictive 2048 game to your mobile device. The objective is to slide numbered tiles on a grid to combine them and create a tile with the number 2048.

## Features

-   **Classic Gameplay**: Authentic 2048 puzzle mechanics.
-   **Smooth Animations**: Enjoy fluid tile movement and merging animations.
-   **High Score Tracking**: Keeps track of your best performance.
-   **Responsive Design**: Optimized for different screen sizes and orientations.
-   **Clean UI**: Minimalist interface for distraction-free gaming.

## Technologies Used

-   **Language**: Kotlin
-   **Framework**: Android SDK
-   **Build System**: Gradle (Kotlin DSL)
-   **Architecture**: Native Android development

## Setup Instructions

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/zoro-00/2084-game.git
    ```
2.  **Open in Android Studio**:
    -   Launch Android Studio.
    -   Select **Open an Existing Project**.
    -   Navigate to the cloned directory and select it.
3.  **Build the Project**:
    -   Allow Gradle to sync and download dependencies.
    -   Click on **Build > Make Project**.
4.  **Run the App**:
    -   Connect an Android device or start an emulator.
    -   Click on **Run > Run 'app'**.
